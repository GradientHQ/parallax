# Parallax
* Dynamic KV Cache Management + Continuous Batching for MAC;
* Pipeline Parallel sharding;


# Install
- MAC: pip install -e '.[mac]'
- GPU: pip install -e '.[gpu]'

- Extra step for development: pip install -e '.[dev]'
